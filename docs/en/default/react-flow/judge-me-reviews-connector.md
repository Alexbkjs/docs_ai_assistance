# ✔️ What is [Judge.me](https://Judge.me) Review?

  

[Judge.me](https://Judge.me) Reviews is a Shopify application that allows users to display review widgets, collect, manage, and display customers’ reviews.

  

# ✔️ How to connect [Judge.me](https://Judge.me) with React Flow?

  

****Step 1.**** Install the [<u><span style="white-space:pre-wrap" class="csh-markdown csh-markdown-underline">Judge.me Product Reviews App</span></u>](https://apps.shopify.com/judgeme) from the Shopify store.

****Step 2.**** Connect [Judge.me](https://Judge.me) with React Flow

  

1.  Open the React Flow app -> ****Connectors tab**** -> click on [****Judge.me****](https://Judge.me).
2.  (optional) Enter the name and description of the connector.
3.  Click ****Sign in to**** [****Judge.me****](https://Judge.me) -> ****Authorize****.
4.  Click on Test if you want to test this connection.
