# Connect the Judge.me Review App

## What is [Judge.me](https://Judge.me) Review?

[Judge.me](https://Judge.me) Reviews is a Shopify application that lets users display review widgets, collect, manage, and showcase customer reviews.

## How to connect [Judge.me](https://Judge.me) with React Flow?

**Step 1.** Install the [<u><span style="white-space:pre-wrap" class="csh-markdown csh-markdown-underline">Judge.me Product Reviews App</span></u>](https://apps.shopify.com/judgeme) from the Shopify App Store.

**Step 2.** Connect [Judge.me](https://Judge.me) with React Flow.

1.  Open the React Flow app → **Connectors tab** → click on [**Judge.me**](https://Judge.me).
2.  (optional) Enter the name and description of the connector.
3.  Click **Sign in to** [**Judge.me**](https://Judge.me) → **Authorize**.
4.  Click **Test** if you want to verify this connection.
