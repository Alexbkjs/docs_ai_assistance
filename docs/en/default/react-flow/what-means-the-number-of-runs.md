# What means the “number of runs”?

  

# Number of runs

  

Number of Runs: This refers to the total count of times each individual workflow is executed.

  

For example, if you want to assign a VIP tag to customers who purchase goods worth more than $1,000, the flow will execute each time a customer places an order to determine if the VIP tag should be added. Each execution counts as one run.
