# Number of Runs

The number of runs refers to the total count of times each individual workflow is executed.

For example, if you want to assign a VIP tag to customers who purchase goods worth more than $1,000, the workflow executes each time a customer places an order to determine whether the VIP tag should be added. Each execution counts as one run.
