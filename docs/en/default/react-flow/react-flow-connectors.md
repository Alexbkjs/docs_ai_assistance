# React Flow Connectors

# What are Connectors?

**Connector** - an option that allows you to connect to the required services. All connectors are available in the Connectors tab in the left column inside the app.

![](https://cdn.heymantle.com/docs/screenshots/e7b00cc6-c403-44af-95d2-ddfefa3d62dd/connectors-list_50es49.png)
