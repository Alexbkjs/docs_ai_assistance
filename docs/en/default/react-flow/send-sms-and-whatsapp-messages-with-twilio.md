# ✔️ What is Twilio?

Twilio - a communication service that provides automatically generated phone numbers that you can use in your business to communicate with customers.

# ✔️ Benefits of using Twilio in React Flow

React Flow provides several communication channels with customers: email, SMS, and WhatsApp. Private messages are one of the most comfortable and fastest communication channels. Therefore, in order to provide our customers with the ability to send SMS, we have chosen the Twilio service.

*   ****New communication channels.**** People usually respond faster to SMS so we decided to expand the number of communication channels with customers. Now you can communicate with customers via SMS and WhatsApp messages.
*   ****Easy to integrate.**** To connect Twilio and React Flow you need just to register in the Twilio service and copy the appropriate data to the React Flow. After setting up and connecting to Twilio, you will immediately have access to actions for sending SMS and WhatsApp messages.
*   ****High-quality connections.**** According to the SLA (Service Level Agreement), Twilio’s software is rated as 99,95%. That means that the connection quality of the service is good and your messages will reach the addressee on time.
*   ****User security.**** Personal data security is always our priority. We guarantee the security of the data you use in the application. Twilio also protects users' data securely and provides two-factor authorization.

# ✔️ How to connect Twilio to React Flow?

1.  ****Twilio - Generate phone number for SMS****

To connect Twilio to React Flow you need to have an account in the Twilio service. If you have account, log in and skip this step. If you don’t have it, sign up the [Twilio](https://www.twilio.com/).

Log in to your account. On the admin panel you will get Account SID and Auth Token.

![](https://cdn.heymantle.com/docs/screenshots/e77fd4d7-fdc2-497d-8bd4-df0c1534a125/get-sid-and-auth_9zzhi7.png)

Click on Get a Twilio phone number and the service will automatically generated you a phone number. You will need this data in React Flow.

Note: In the React Flow you can only use the numbers you entered in Twilio.

2.  ****Twilio - Connect WhatsApp****

If you want to register a number for brand, you should follow the instruction from the article [WhatsApp Self Sign-up](https://www.twilio.com/docs/whatsapp/self-sign-up).

If you want to register a number not for brand, you must fill out Twilio’s [Request Access](https://www.twilio.com/en-us/whatsapp/request-access) form. You need to provide the requested information and wait for confirmation from Twilio. Please review the [Getting Started](https://www.twilio.com/docs/whatsapp/tutorial/requesting-access-to-whatsapp) guide for details.

3.  ****React Flow - Create connector****

Open the React Flow app and go to the Connectors tab and click on Twilio connector.

Step 1. (optional) Enter the name and description of the connector.

Step 2. In your Twilio account copy Account SID and Auth Token and enter them in the appropriate fields in React Flow.

Step 3. In the upper right corner, you can find the Test button - it is designed to check if the connector works correctly. Click on it and enter a phone number generated by Twilio and a test phone number where you will send a test message. You will get SMS with confirmation.

![](https://cdn.heymantle.com/docs/screenshots/5bafca3a-6a7a-4682-92f4-954207e5380a/connector-twilio_hf1hs0.png)

Step 4. Click Save on the upper right corner

Now, if you go to the Connectors, you will see that Twilio connector’s status is connected. As soon as you have connected the Twilio, two actions become available to you in the React Flow application: Send SMS and Send WatsApp message actions.

# ✔️ Send SMS action

Send SMS action allows you automatically send SMS to your customer.

Example: You want to send a confirmation SMS to customers every time when they paid their order.

1.  Open the React Flow app and click on ****Create new****.
2.  Select trigger ****Order paid****.
3.  Click on ****Add action**** and in the right side menu select ****Send SMS****.
4.  Find the Recipient’s phone filed and click on ****Add variables****.

![](https://cdn.heymantle.com/docs/screenshots/0f448ee3-ccb7-4cc7-96f8-ab0d2797c27f/send-sms-action_10splke.png)

5.  Select the variable ****Customer phone**** and enter your message in the message field.

![](https://cdn.heymantle.com/docs/screenshots/0fa4d17e-6608-4014-9a8c-127f3c7b18e1/select-customer-phone-variable_hr40ax.png)

6.  ****Save**** changes.

Now, when your customer will complete payment, he will receive a confirmation via SMS.

# ✔️ Send WhatsApp message action

Send WhatsApp message action allows you automatically send messages to your customer via WhatsApp.

Example: You want to provide a discount as a welcome bonus for registered customers.

1.  Open the React Flow app and click on ****Create new****.
2.  Select trigger ****Customer created****.
3.  Click on ****Add action**** and in the right side menu select ****Send WhatsApp message****.

![](https://cdn.heymantle.com/docs/screenshots/7da7b891-fa23-48d9-88d1-cdae4dd18aa3/send-whatsapp-message_4saf0b.png)

4.  Find the Recipient’s phone filed and click on ****Add variables****.
5.  Select the variable ****Customer phone**** and enter your message in the message field.

![](https://cdn.heymantle.com/docs/screenshots/fdc36a48-9218-4fed-891d-f7e78864dc74/select-customer-phone-variable_1nkyxsf.png)

6.  ****Save**** changes.

Now, the buyer will receive a message with a discount immediately after registration.
