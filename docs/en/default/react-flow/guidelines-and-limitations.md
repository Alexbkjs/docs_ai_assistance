# React Flow Limitations

## React Flow

**React Flow** is a workflow builder that helps you automate routine tasks and save time for more important things.

The application offers different subscription plans for your convenience. In this article, we'll show you exactly when you need a subscription and which one to choose.

But first, let's recap what **workflows** and the **number of runs** are.

## What is a "workflow"?

A workflow is a chain of triggers, actions, and conditions that helps you set up automated processes for your Shopify store. For example, you can set up automatic emails for new customers with profitable offers, or create a process that reminds you to replenish product stock. React Flow provides many opportunities to simplify your work and save time. Find detailed information in our [guides](https://help.devit.software/en/category/react-flow-2kjdy9/).

![](https://cdn.heymantle.com/docs/screenshots/aa72faec-f0ef-47ed-847b-f08cc8e3084c/all-workflows_u5f8t8.png)

## What is the "number of runs"?

This is the number of times each individual workflow is executed. For example, if you set up a process that automatically sends new registered customers letters with offers, discounts, or coupons, the process runs once for each new customer — each execution counts as one run.

## What will happen if you reach the limit of available runs?

If you reach the limit, your workflow will continue running. However, each subsequent launch over the limit will incur an additional usage fee according to your subscription plan. Note that usage fees apply even during the trial period.

## Timing of workflows

A workflow starts running immediately after activation. You don't need to activate it right away — you can set it up and activate it at any convenient time, and deactivate it whenever needed.

![](https://cdn.heymantle.com/docs/screenshots/a0dc2858-7311-4f6d-a5f5-cb5c1b9aba4b/turn-on-workflow_1dyrnov.png)

## Delay actions

One of the main advantages of React Flow is the ability to **delay actions**.

For example, you may want to send customers an email when they delete an order — but not immediately; instead, after 3 hours.

Select the **Order deleted** trigger and click **Add delay**. Select the type and value (3 hours). Then select the **Send email** action and fill in the appropriate text. Your email will be sent to the customer 3 hours after the order is deleted.

## What are the different subscription plans for?

Some owners create a few workflows and run them 1–2 times a month, while others fully automate their store and use them daily. There are plans with more features for higher-volume needs.

**Pay as you go:** If you have a small store and need a limited number of features, try this plan. It allows a maximum of 20 active workflows and 100 runs per month. There are no subscription charges, but if you exceed the monthly limit of 100 runs, you'll be charged per extra run.

React Flow also offers the following plans:

1.  **Pay as you go (free to install)**
2.  **Starter**
3.  **Standard**
4.  **Professional**
5.  **Unlimited**
6.  **Enterprise**

All of the above plans include a certain number of available workflows and runs. If you exceed this amount, you'll be charged per extra run over the monthly limit.

All plans include a 7-day free trial period. The trial lets you test paid subscriptions with extended limits. Usage charges may apply during the trial period if the automation run limit is exceeded.

**Unlimited plan:** You get unlimited workflows, unlimited runs per month, and dedicated support.

**Enterprise:** This plan offers personal terms and contracts tailored to your company.

You can choose any plan based on your needs. Plans differ in the number of workflows, runs, and overage charges. View detailed information inside the React Flow app under the Plans section, on the [App listing page](https://apps.shopify.com/react-flow), or on our [Official website](https://devit.software/react-flow).

The app has no additional payments — extra charges only apply if you exceed the workflow limit, according to your subscription plan. Payment is charged every 30 days.

## Tips for shop owners

**Flow Creation Tip:** Combine tasks into a single flow when possible. For instance, instead of creating 5 separate flows that each check **order created** → **product tag condition** → **action**, handle them all in one flow.

It's possible to use 1 flow with 5 conditions, so each created order counts as only 1 run instead of 5.

✅ If you need to create many workflows and run a lot of automations, and the free version of the app is no longer sufficient, you have the option to change your subscription. Choose one of the plans and free up more time for other tasks while **React Flow** works for you.
