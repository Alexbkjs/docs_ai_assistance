# Can I get a demo of the app?

# Get A Demo

We provide a free trial period when you can test all features for free.

Note that if you exceed the run limit, usage fees will be charged even during the trial period. After the trial period ends, you will be charged for the first month. If you uninstall the application before the trial period concludes, no charges will apply. Additionally, you can book a Demo of the application at any time. Our manager will call you at the agreed time and gladly show you the application.
