# Single Activity page

# What is the single activity page?

Once you've created specific workflows, you can observe how each process works independently.

Click on any workflow, for example, **Order created**. After this, you'll see a new page. It's a **Single Activity page**. Here, you can find detailed information about the created workflow and how it functions.

![](https://cdn.heymantle.com/docs/screenshots/64eeb080-7752-42b9-8e27-493346ae2ce9/single-activity-page_15c6qrd.png)

# How can a single activity page be useful?

If you have questions about workflow settings or need clarification on why certain actions occurred, you can find the information here.

Let's follow an example.

In this workflow, three conditions were evaluated to assign a tag to customers. Here, you can identify which condition was met.

Green-highlighted squares indicate the conditions that were fulfilled. In this example, we can see that the buyer made a purchase exceeding 750, triggering the assignment of the **"Gold"** tag.

![](https://cdn.heymantle.com/docs/screenshots/17272baa-1e1c-44b7-bc19-f3b77bd775da/snimok-ekrana-2022-05-18-v-094_1iqr25k.png)

Respectively, the gray-highlighted squares indicate that these conditions were not met in the workflow.

![](https://cdn.heymantle.com/docs/screenshots/7c2087f1-3443-4e10-b135-89253a582a27/snimok-ekrana-2022-05-18-v-094_14loil.png)

On the bottom of the page, all the same information is available in text. The order status is marked as "fulfilled" and condition of more than 750 is met.

![](https://cdn.heymantle.com/docs/screenshots/0e518b24-c481-4f23-a4b6-81a7d84993c6/snimok-ekrana-2022-05-18-v-100_vnjbq.png)

The buyer was assigned a **"Gold"** tag.

![](https://cdn.heymantle.com/docs/screenshots/79934c84-6408-4af6-a758-868fa3f1556b/snimok-ekrana-2022-05-18-v-100_1e0ygic.png)

If you found an error in the workflow on a **Single Activity page**. It can be fixed on the **"Workflow"** tab.

# Final result

To sum it up, on the **Single activity page**, store owner can check workflow results as well as find out all the necessary details.
