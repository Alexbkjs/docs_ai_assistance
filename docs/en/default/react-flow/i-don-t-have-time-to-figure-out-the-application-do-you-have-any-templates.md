# I don't have time to figure out the application. Do you have any templates?

# React Flow Templates

Yes, we have. We value your time, so we have created ready-made templates of the most popular flows.

To launch a template you should go to the ****Templates section****, select the appropriate template, and launch it.

For example, react Flow provides a template that tags orders with foreign currencies. The application verifies the payment currency of each order. If the currency is not USD, the application automatically applies a tag to the order.

In the templates list, you will also find the following workflows: Delayed reminder of unfulfilled orders, High-risk order alert, Marketing tags, etc.
