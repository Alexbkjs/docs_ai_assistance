# React Flow Templates

Yes, we do. We value your time, so we've created ready-made templates for the most popular flows.

To launch a template, go to the **Templates** section, select the appropriate template, and enable it.

For example, React Flow provides a template that tags orders with foreign currencies. The app checks the payment currency of each order. If the currency is not USD, it automatically applies a tag to the order.

In the templates list, you'll also find workflows such as: Delayed reminder of unfulfilled orders, High-risk order alert, Marketing tags, and more.
