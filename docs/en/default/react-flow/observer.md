# ✔️ What is the Observer feature?

  

Observer is used to ****track the updates**** within customers, orders, products, etc. This option will help you to track updates in the product’s title, tags, customer’s name, etc.

  

Observer helps you ****stay informed about important changes**** such as updates to product titles or tags, customer name modifications, and other critical adjustments. With this option, you can easily keep an eye on what’s happening in your store without manual checks, ensuring you never miss an important update.

  

# ✔️ How to track updated items?

  

****Example:**** Tracking the addition of new tags with the required value of products.

  

1.  Select the trigger ****Order updated****.
2.  Click ****Select**** in the right-side menu -> ****Order**** -> ****Tags****.

  

![](https://cdn.heymantle.com/docs/screenshots/ccadb0cf-c25e-4524-a50a-ac5d0fcf0788/select-condition-order-updated_2fkw4e.png)

  

3.  Click on the + icon in the trigger block and select Condition.
4.  On the right-side menu, click on the condition field.
5.  Scroll until you find the ****Observe**** option.
6.  Click on it and select ****Added items****.

  

![](https://cdn.heymantle.com/docs/screenshots/df2c5015-81e6-4619-ac83-e9b6431499a5/observe-added-items_1io78ew.png)

  

7.  Click ****Select matcher**** -> is equal -> set the required value.

  

![](https://cdn.heymantle.com/docs/screenshots/5438843b-3091-49d7-a2ee-4c91908e242d/set-value_2yuuk4.png)

  

You can add more conditions or actions to the flow if needed. When you’ve finished with the configuration, just click ****Save**** in the upper-right corner.
