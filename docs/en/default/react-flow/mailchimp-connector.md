# Connect Mailchimp

## What is Mailchimp?

[Mailchimp](https://mailchimp.com/) is a marketing platform primarily used for email marketing. It helps businesses create and send emails, manage contact lists, automate campaigns, and track performance metrics like opens and clicks. It also supports basic SMS marketing, landing pages, and audience analytics.

## How to connect Mailchimp with React Flow?

### Step 1: Copy API key in Mailchimp

If you already have an API key in your Mailchimp account, copy it and proceed to Step 2.

If you don't have an API key, generate one:

1.  Open the Mailchimp [dashboard](https://us22.admin.mailchimp.com/account/api/).
2.  Scroll to the **"Your API keys"** section and click **Create a Key**.
3.  Enter the key name and click **Generate Key**.

![](https://cdn.heymantle.com/docs/screenshots/5b1190aa-446c-4c6d-84d6-55440d06bbc3/mailchimp-api-name_1ffvx9l.png)

4.  Copy the generated key.

![](https://cdn.heymantle.com/docs/screenshots/dd00436e-11d5-439b-aff6-03d9bf9253ab/mailchimp-generated-key_7t6rt2.png)

### Step 2: Connect Mailchimp with React Flow

1.  Open the React Flow app → **Connectors tab** → click on **Mailchimp**.
2.  (optional) Enter the name and description of the connector.
3.  Enter your API key in the appropriate field.
4.  Click **Test** if you want to verify this connection.

![](https://cdn.heymantle.com/docs/screenshots/867b22cb-0ed7-4ce4-92e9-52a1ee14c665/mailchimp-connector_j5xag4.png)
