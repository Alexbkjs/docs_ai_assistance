# Connect Slack

## What is Slack?

Slack is a team communication tool that lets people chat in channels, send direct messages, share files, and integrate with other work apps. It is a real-time hub for team collaboration.

## How to connect Slack with React Flow?

1.  Open the React Flow app → **Connectors tab** → click on **Slack**.
2.  (optional) Enter the name and description of the connector.
3.  Click **Add to Slack** and authorize your account.
4.  Click **Test** if you want to verify this connection.
