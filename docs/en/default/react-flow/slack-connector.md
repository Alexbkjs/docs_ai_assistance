# ✔️ What is Slack?

  

Slack is a team communication tool that lets people chat in channels, send direct messages, share files, and integrate with other work apps. It’s basically a real-time hub for team collaboration.

  

# ✔️ How to connect Slack with React Flow?

  

1.  Open the React Flow app -> ****Connectors tab**** -> click on ****Slack****.
2.  (optional) Enter the name and description of the connector.
3.  Click ****Add to Slack**** and authorize to your account.
4.  Click on Test if you want to test this connection.
