# Automated Processes

The app provides multiple triggers and actions, allowing you to automate many processes.

For example, you can add marketing tags, send daily reports with low-stock items, send high-risk order alerts, or send unfulfilled order reminders to the store owner.

For more information, refer to the relevant articles: [Order Workflows](/react-flow/order-workflows), [Customer Workflows](/react-flow/customer-workflows), [Product Workflows](/react-flow/product-workflows).

We have also collected the [most popular workflows](/react-flow/most-popular-flows) in one article. Check it out to learn more about the app's features.
