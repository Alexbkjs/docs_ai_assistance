# What processes can I automate with the help of your app?

  

# Automated Processes

  

The app provides multiple triggers and actions, so you can automate many processes.

  

For example, you can add marketing tags, send everyday reports with low-stock items, inform about high-risk orders, send a reminder about unfulfilled orders to the store owner, etc.

  

For more information please refer to the relevant articles: [Order Workflows](/order-workflows), [Customer Workflows](/customer-workflows), [Product Workflows](/product-workflows).

  

We have also collected the [most popular workflows](/most-popular-flows) in one article. Check it out to learn more about the app's features.
