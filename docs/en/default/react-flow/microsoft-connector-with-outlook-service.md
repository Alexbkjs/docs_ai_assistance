# ✔️ What is a Microsoft connector?

Microsoft is a new connector that allows you to use "Microsoft Outlook" as a provider for sending email from the "Send email" action.

# ✔️ How to activate the Microsoft connector?

Enter the React Flow app -> Connectors tab.

1.  In the Connectors section, select ****Microsoft****.

![](https://cdn.heymantle.com/docs/screenshots/85b419b0-b9a4-47ce-9e93-5ac02503d508/select-new-connector_1rdowo2.png)

2.  Enter the ****Name**** and ****Description**** of the connector.
3.  Click on Sign in with Microsoft and complete the registration.
4.  In the upper right corner click on the ****Test button**** - it is designed to check the correct information. Please use it to ensure that you have entered the correct information.

![](https://cdn.heymantle.com/docs/screenshots/655d9fbb-a3ff-49f5-abc9-94b185a46262/configure-connector_p5rx63.png)

5.  ****Save**** configuration.

# ✔️ Send email action

Send email action allows you to automatically send emails to your customers.

Example: a customer made the first order and you want to provide him a 10% discount.

****Add existing connector****

1.  Select trigger ****Order created.****
2.  Click on 'then' and click on ****Action****.
3.  Find the ****Send email**** action in the Connectors section and click on it.

![](https://cdn.heymantle.com/docs/screenshots/8ff85cd4-37ff-440b-9868-38016a319215/send-email-connectors_8ut5dh.png)

****Note:**** you can also choose the Send Email action from the External section. By default, React Flow is connected, but you can select your Microsoft connector instead. \*\*

4.  In the Connection section you can add your Microsoft connector.

![](https://cdn.heymantle.com/docs/screenshots/b9b3f499-a027-4acc-a9c9-f5b135bda998/add-the-required-connector_1kfjjjb.png)

5.  Fill in the remaining fields with the required information and ****Save**** your workflow in the upper right corner.
6.  Turn on workflow in the upper right corner.

****Create a new connector****

If you don't have connector or you want to create a new one, follow these steps:

1.  In the Connection section click ****Change**** -> ****Create new****
2.  Click on ****Microsoft**** in the pop-up window.
3.  Enter Name, Description and click ****Sign in with Microsoft****.
4.  Click ****Save****.

![](https://cdn.heymantle.com/docs/screenshots/050691c1-52e1-4d2f-9d00-30a0a9d6cdc2/create-new-connector_173rqgo.png)
