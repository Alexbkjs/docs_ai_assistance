# ✔️ About the Message reply option

The Message Reply Option lets you control whether recipients can reply to the email message. This setting also creates a history of correspondence in the channel, allowing you to track the entire email conversation directly within the channel.

*   ****When enabled****, users can reply directly to the email message from Discord, making it easy to continue the conversation.
*   ****When disabled****, replies are turned off - perfect for announcements or one-way updates where no response is needed.

![](https://cdn.heymantle.com/docs/screenshots/794a6862-6f94-4bec-9294-0d11a7a27c6a/reply-option_1992gkl.png)

Use this setting to customize how interactive your email communication should be.

# ✔️ Activate the Message reply option

1.  Use the `/list` command in the channel connected to the Email bot.
2.  Click Settings and select ****Message Reply Option****.
3.  Click ****Enable**** to activate the option.
4.  Send a test message to your channel to check the option.
