# Message reply option

## About the message reply option

The Message Reply Option lets you control whether recipients can reply to email messages. It also creates a correspondence history in the channel, allowing you to track the entire email conversation directly in Discord.

*   **When enabled**, users can reply directly to the email message from Discord, making it easy to continue the conversation.
*   **When disabled**, replies are turned off — ideal for announcements or one-way updates where no response is needed.

![](https://cdn.heymantle.com/docs/screenshots/794a6862-6f94-4bec-9294-0d11a7a27c6a/reply-option_1992gkl.png)

Use this setting to control how interactive your email communication should be.

## Activate the message reply option

1.  Use the `/list` command in the channel connected to the Email bot.
2.  Click Settings and select **Message Reply Option**.
3.  Click **Enable** to activate the option.
4.  Send a test message to your channel to verify the option works.

In the Settings section, you can also customize:

*   [Web Version of Emails](/email/web-version-of-emails)
*   [Email Preview in Message](/email/email-preview-in-message)
*   [Email Personalization](/email/email-personalization)
