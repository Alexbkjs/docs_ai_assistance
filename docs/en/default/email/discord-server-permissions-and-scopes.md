# Required Discord server permissions and scopes

## Check bot permissions

This article covers the following required permissions:

*   **Manage Webhooks** – allows the bot to create and delete webhooks used to send emails to your Discord channel. To enable this permission:

1.  Open Server Settings.
2.  Go to Roles.
3.  Click the bot name or click Edit next to the bot's name.
4.  Go to Permissions.
5.  Enable the Manage Webhooks option.

![](https://cdn.heymantle.com/docs/screenshots/8b742736-aff3-4c11-9bac-645103b8b9b9/permission-manage-webhook_1mzf1uj.png)

*   **View Audit Log** – allows the bot to identify who added it, so we can send them important Email Bot updates. To enable this permission:

1.  Open Server Settings.
2.  Go to Roles.
3.  Click the bot name or click Edit next to the bot's name.
4.  Go to Permissions.
5.  Enable the View Audit Log option.

![](https://cdn.heymantle.com/docs/screenshots/698e292c-df96-42d4-a275-68cb76aa8864/permission-view-audit-log_13wsq0x.png)

**Missing permissions?** Permissions are critical for the bot connection process and correct operation. If you do not have the required permissions, you will receive a message in the channel immediately after running the `/connect` command.

![](https://cdn.heymantle.com/docs/screenshots/e6467408-a910-4809-9d60-8bda0826eae9/connection-failed_1r4j7xv.png)
