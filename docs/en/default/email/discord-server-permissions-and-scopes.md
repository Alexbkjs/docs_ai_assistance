# ✔️ Check Bot permissions

  

In this article, we will review the following required permissions:

  

*   ****Manage Webhooks permissions**** - it allows to create and delete webhooks that are used to send emails to your Discord channel. You can check these permissions by following these steps:

1.  Open Server Settings.
2.  Go to Roles.
3.  Click on the bot name or on Edit near the bot’s name.
4.  Go to Permissions.
5.  Enable the Manage Webhooks option.

  

![](https://cdn.heymantle.com/docs/screenshots/8b742736-aff3-4c11-9bac-645103b8b9b9/permission-manage-webhook_1mzf1uj.png)

  

*   ****View Audit Log permission**** - this option lets us identify who added the bot, so we can send them important Email Bot updates. You can check these permissions by following these steps:

  

1.  Open Server Settings
2.  Go to Roles.
3.  Click on the bot name or on Edit near the bot’s name..
4.  Go to Permissions.
5.  Enable the View Audit Log option.

  

![](https://cdn.heymantle.com/docs/screenshots/698e292c-df96-42d4-a275-68cb76aa8864/permission-view-audit-log_13wsq0x.png)

  

Missing Permissions?Permissions are critical in the bot connection process and for its correct operation. If you do not have enough permissions, you will receive a message in the channel where the bot connects immediately after executing the `/connect` command.

  

![](https://cdn.heymantle.com/docs/screenshots/e6467408-a910-4809-9d60-8bda0826eae9/connection-failed_1r4j7xv.png)
