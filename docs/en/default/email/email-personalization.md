# ✔️ About Personalization settings

  

Personalization Settings let you customize how your emails appear in Discord, helping you match your communication style or brand identity.

  

With this option, you can:

  

*   Set a custom bot name to replace a sender's name and email address.
*   Choose whether to display the sender’s avatar or a custom logo for a more recognizable look.

  

How it works:

  

*   If you enter a custom bot name, it will be shown in place of the sender name and address.
*   If you leave it blank, the sender's name and email address will be used.

  

This is a great way to make your messages feel more personal - or more professional - depending on your needs.

  

# ✔️ Set the sender’s avatar

  

1.  Use the `/list` command in the channel connected to the Email bot.
2.  Click Settings → Select Settings and select ****Email Personalization****.
3.  Near the Show avatars by Email click ****Enable**** to activate the option.

  

Now you can add avatar to the bot:

  

1.  Edit channel → Integrations → Webhooks.
2.  Click on the Email bot and change the avatar.
3.  Send a test message from your email service to check if the option works correctly.

  

Note: To use the Webhook Avatar in your message, please enable the ****Manage Webhooks**** permission: go to Server Settings → Roles → Email → Permissions → Enable “Manage Webhooks.” After that, update the avatar again (remove and re-add it).

  

# ✔️ Set the custom bot name

  

1.  Use the `/list` command in the channel connected to the Email bot.
2.  Click Settings → Select Settings and select ****Email Personalization****.
3.  Near the Custom Bot Name click ****Edit****.
4.  In the appeared window enter the name you want to display and click ****Submit****.
5.  Send a test message from your email service to check if the option works correctly.
