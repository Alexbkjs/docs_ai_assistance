# Web version of emails

## About the option

The Web Version of Emails setting lets you include a link to a web-based version of the email directly within the main message.

*   **When enabled**, each email will include a link to a web-based version that can be opened in any browser — helpful for sharing or viewing messages outside of Discord.
*   **When disabled**, the email will only appear in its original format within the channel.

![](https://cdn.heymantle.com/docs/screenshots/5bf0eeb0-ca5f-4d8e-847e-25391333748b/web-version-of-email_d6chfr.png)

This option gives you flexibility in how you access and manage your email content.

## Activate the web version of emails option

1.  Use the `/list` command in the channel connected to the Email bot.
2.  Click Settings → Select Settings and select **Web version of emails**.
3.  Click **Enable** to activate the option.
4.  Send a test message to verify the option works correctly.

In the Settings section, you can also customize:

*   [Email Preview in Message](/email/email-preview-in-message)
*   [Message Reply Option](/email/message-reply-option)
*   [Email Personalization](/email/email-personalization)
