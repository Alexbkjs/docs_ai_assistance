# ✔️ About Email preview in Message

  

The Email Preview in Message option allows you to include a preview of the email content directly within the main message.

  

*   ****When enabled****, a preview of the email body appears directly in the message - giving you a quick look without needing to open anything.
*   ****When disabled****, only the email subject is displayed, keeping the message cleaner and more compact.

![](https://cdn.heymantle.com/docs/screenshots/79a068fd-352e-4007-baf6-2041bacfa7ae/email-preview_1ykn8o5.png)

  

  

This setting lets you balance visibility and privacy. If you’d prefer other channel members not to see the full email content, simply enable this option.

  

# ✔️ Activate the option

  

1.  Use the `/list` command in the channel connected to the Email bot.
2.  Click Settings -> Select Settings and select ****Email Preview in Message****.
3.  Click ****Enable**** to activate the option.
4.  Send a test message to check if the option works correctly.

  

Note: If you disable the Email Preview in Message option but the message body doesn’t appear in Discord, your email provider may be sending messages in HTML format by default. To fix this, check your email provider’s settings and choose Plain Text format for the message.
