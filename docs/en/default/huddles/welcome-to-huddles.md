## What is Huddles

[Huddles](https://top.gg/bot/1123558266555871252) is a Discord bot that lets you start a group audio conversation in any channel.

Discord is a great communication platform, but starting a group call requires several steps. That's where Huddles comes in — it makes the process fast and simple.

## How does it work

Invite Huddles to your server and start a call in any channel using the `/start` command. All channel members can join the call by following the generated link.

![Start a call in one click](https://cdn.heymantle.com/docs/screenshots/a579674e-ca05-423d-8783-69312ec2fb39/lets-make-a-call_cb57ma.png)

![Multi-person meeting in Huddles](https://cdn.heymantle.com/docs/screenshots/bbf2934c-1c71-4676-8839-d44fc374d774/meeting_3l0w64.png)

Video, multi-person screen sharing, a dedicated thread for notes, and emoji reactions are all available during calls.

After the call ends, the bot automatically deletes the voice channel.
