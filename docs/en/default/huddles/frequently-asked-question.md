# ✔️ Can I share my screen and use camera while using Huddles?

Yes, all these options are still available for you.

# ✔️ Does the bot records the call?

No, Huddles doesn’t record calls.

# ✔️ How does the Huddle call work?

The bot creates a temporary voice channel that all channel members can enter.

# ✔️ Do I need to remove the voice channel manually?

No, the voice channel will be deleted automatically at the end of the call when everyone leaves the channel or after 5 minutes if no one has entered.

# ✔️ Can I use Huddles in any channel of my server?

Yes, you can use Huddles in any channel. Just invite the bot and follow instructions from this article.

# ✔️ If I need help with Huddles, how can I get it?

If you have any question, please refer to our support team: [support@devit.software](mailto:support@devit.software).

# ✔️ Where can I leave feedback about the bot?

You can leave your feedback on the [Huddles](https://top.gg/bot/1123558266555871252) page.
