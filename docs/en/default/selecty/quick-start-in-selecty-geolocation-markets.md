# ✔️ About the Selecty application

The application will be helpful for stores that want to simplify the user experience of the store for customers in the international market. You can enable markets redirect to set the visitor's country and currency automatically according to the IP geolocation and the language according to the browser's preferred language. Enable recommendation banner to use geolocation detection and personalized language and currency suggestions for your customers.

You can also configure selectors so that users can manually toggle language, country, currency, or all-at-once switchers.

# ✔️ How to customize the recommendation banner?

We suggest starting work on the application by setting up a recommendation banner because that's the first thing the visitors will see at their first visit to the store.

1.  ****Step 1.**** Scroll to the ****Markets Recommendations**** section and click ****Customize****.

![](https://cdn.heymantle.com/docs/screenshots/0c59d858-e4de-4ec6-a148-0bf606f48c4d/customize-banner_gk8ab3.png)

Language and currency must be added in advance in your store settings.

2.  ****Step 2.**** Set the [markets display recommendation option](/markets-recommendation-display-option) - select when the banner should be displayed to visitors.
3.  ****Step 3.**** Next, choose the type of the recommendation banner.
4.  ****Step 4.**** Choose one of the [available resources](/available-resources). For example, ****Language****.

![](https://cdn.heymantle.com/docs/screenshots/b4061bbc-1280-45ef-9488-8198e6002c4d/display-option-type-resource_11lujds.png)

5.  ****Step 5.**** You can also customize the resource view, color scheme, scale, and animation.

![](https://cdn.heymantle.com/docs/screenshots/cc660efe-2330-4fb7-a46a-c8e992e8add9/banners-view_cr6tuu.png)

6.  ****Step 6.**** ****Save**** the banner settings with the button in the upper right corner.

# ✔️ How to enable markets redirect?

Set the visitor's country and currency automatically according to the IP geolocation and the language according to the browser's preferred language in one click. All you need is to enable the appropriate options in the Markets redirect section.

![](https://cdn.heymantle.com/docs/screenshots/21106c74-d9a1-46a1-9dad-2393ca591acb/markets-redirect_106g1xf.png)

# ✔️ How to customize selectors?

You have an opportunity to create currency, language, and country selectors. You can also customize any selector according to the design of your store.

1.  ****Step 1.**** Click on the ****Create selector**** or ****Create new**** button.

![](https://cdn.heymantle.com/docs/screenshots/3b3255e3-4433-4bf9-9453-419d2b004a75/create-selector_3mhk2h.png)

2.  ****Step 2.**** In this step, choose one of the themes that best suits your store.

![](https://cdn.heymantle.com/docs/screenshots/a597ddfe-e076-4296-b681-21210591fb79/select-theme_1n15vtn.png)

3.  ****Step 3.**** Choose one of the [available resources](/available-resources). For example, ****Language & Currency****.
4.  ****Step 4.**** You can also change the type of the selector.

![](https://cdn.heymantle.com/docs/screenshots/5179529e-fe88-4a69-81d0-bd5403a0c12a/resource-and-type_r8s87b.png)

5.  ****Step 5.**** Now, choose a color scheme that suits your store.
6.  ****Step 6.**** Adjust the position of the selectors, display icons, names format and search.

![](https://cdn.heymantle.com/docs/screenshots/01a4d170-cf69-4510-b74c-99c55f23fd6a/color-scheme-and-position_1usdko.png)
