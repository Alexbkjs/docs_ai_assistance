# Quick start in Geolocation & Markets

## About the application

This application helps stores simplify the shopping experience for international customers. You can enable markets redirect to automatically set a visitor's country and currency based on IP geolocation, and their language based on the browser's preferred language. You can also enable the recommendation banner to use geolocation detection and provide personalized language and currency suggestions.

Additionally, you can configure selectors so that users can manually switch language, country, currency, or all at once.

## How to customize the recommendation banner

We recommend starting with the recommendation banner, as it is the first thing visitors see when they arrive at your store.

1.  **Step 1.** Scroll to the **Markets Recommendations** section and click **Customize**.

![](https://cdn.heymantle.com/docs/screenshots/0c59d858-e4de-4ec6-a148-0bf606f48c4d/customize-banner_gk8ab3.png)

Language and currency must be added in your store settings in advance.

2.  **Step 2.** Set the [markets display recommendation option](/selecty/markets-recommendation-display-option) — choose when the banner should appear to visitors.
3.  **Step 3.** Choose the type of the recommendation banner.
4.  **Step 4.** Choose one of the [available resources](/selecty/available-resources), for example **Language**.

![](https://cdn.heymantle.com/docs/screenshots/b4061bbc-1280-45ef-9488-8198e6002c4d/display-option-type-resource_11lujds.png)

5.  **Step 5.** Customize the resource view, color scheme, scale, and animation.

![](https://cdn.heymantle.com/docs/screenshots/cc660efe-2330-4fb7-a46a-c8e992e8add9/banners-view_cr6tuu.png)

6.  **Step 6.** **Save** the banner settings using the button in the upper right corner.

## How to enable markets redirect

Set a visitor's country and currency automatically based on IP geolocation, and their language based on the browser's preferred language — all in one click. Simply enable the appropriate options in the **Markets redirect** section.

![](https://cdn.heymantle.com/docs/screenshots/21106c74-d9a1-46a1-9dad-2393ca591acb/markets-redirect_106g1xf.png)

## How to customize selectors

You can create currency, language, and country selectors, and customize each one to match your store's design.

1.  **Step 1.** Click **Create selector** or **Create new**.

![](https://cdn.heymantle.com/docs/screenshots/3b3255e3-4433-4bf9-9453-419d2b004a75/create-selector_3mhk2h.png)

2.  **Step 2.** Choose a theme that best suits your store.

![](https://cdn.heymantle.com/docs/screenshots/a597ddfe-e076-4296-b681-21210591fb79/select-theme_1n15vtn.png)

3.  **Step 3.** Choose one of the [available resources](/selecty/available-resources), for example **Language & Currency**.
4.  **Step 4.** Select the type of selector.

![](https://cdn.heymantle.com/docs/screenshots/5179529e-fe88-4a69-81d0-bd5403a0c12a/resource-and-type_r8s87b.png)

5.  **Step 5.** Choose a color scheme that suits your store.
6.  **Step 6.** Adjust the position of the selectors, display icons, name format, and search.

![](https://cdn.heymantle.com/docs/screenshots/01a4d170-cf69-4510-b74c-99c55f23fd6a/color-scheme-and-position_1usdko.png)
