# Can I customize selectors?

  

# Customization

  

Sure! The app provides multiple options for customization.

  

When you create a new selector, you'll be directed to the customization page. There you can select ****theme, type, and color scheme**** of your selector. To offer you the best services, we are constantly working on creating new types of themes and color schemes.

  

# ✔️ Available themes

  

*   ****Basic**** - provides a standard view of buttons and a simple and intuitive interface.
*   ****Material**** - a fully responsive theme with a stricter design and colors.
*   ****Cute**** - provides a responsive design with smooth lines and nice color schemes.
*   ****Minimal**** - a theme with a simple and clean design that keeps the focus on your products.

  

# ✔️ Selector Types

  

The selector type determines how the list of languages will unfold on your customer's screen. The Selecty app provides the following types:

  

*   ****Free plan:**** Dropdown, Wheel, Inline, Modal.
*   ****Pro and Premium plans:**** Layered, Sidebar, Cascade, Scale, Tape-line.

# ✔️ Color Schemes

  

The app provides a wide range of color schemes. You can choose a color scheme for each theme separately. You can also instantly apply a particular color scheme and preview how it looks in different themes.

  

![](https://cdn.heymantle.com/docs/screenshots/23bdd81c-55a2-43ed-a36a-bc7e6a6a64c9/selector-color-schemes_xu6dyb.png)

  

Note that you can manually customize colors and fonts for each selector and/or use a custom CSS code.

  

As a result, you will get an attractive banner that will suit your store design and allow visitors to select the correct language or currency.
