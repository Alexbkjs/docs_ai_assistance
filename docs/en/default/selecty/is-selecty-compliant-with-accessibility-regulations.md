# Is Selecty compliant with accessibility regulations?

Yes, Selecty is fully compliant with accessibility regulations, specified in the ****Web Content Accessibility Guidelines**** ([<u><span style="white-space:pre-wrap" class="csh-markdown csh-markdown-underline">WCAG</span></u>](https://www.w3.org/WAI/standards-guidelines/wcag/)) and the ****European Accessibility Act**** ([<u><span style="white-space:pre-wrap" class="csh-markdown csh-markdown-underline">EAA)</span></u>](https://commission.europa.eu/strategy-and-policy/policies/justice-and-fundamental-rights/disability/union-equality-strategy-rights-persons-disabilities-2021-2030/european-accessibility-act_en).

  

When developing selectors and banners, we followed key accessibility principles to ensure an inclusive and user-friendly experience for everyone:

  

*   ****Keyboard accessibility.**** Users can access storefront embedded app elements (selectors, banners, etc) using Tab, Enter and arrow buttons.
*   ****Sufficient color contrast.**** All texts and elements maintain a clear contrast against the background for optimal readability.
*   ****Clear and resizable text.**** App provides a responsive design that adjusts when users zoom and resize the content. We also used clear fonts and sufficient line spacing to make texts well readable.
*   ****Clearly distinguishable interactive elements.**** Links, buttons, and controls are visually distinct from non-interactive elements, helping users recognize clickable areas quickly.
*   ****Alternative text and captions****. All non-text elements include alternative text, improving accessibility for users with visual impairments.
*   ****Responsive design.**** Storefront embedded app elements (selectors, banners, etc) automatically adapt to different devices, screen sizes, and zoom levels.

  

This improvement makes Selecty more inclusive and accessible, providing equal access to everyone and meeting modern web accessibility regulations set by law in multiple countries all over the world.
