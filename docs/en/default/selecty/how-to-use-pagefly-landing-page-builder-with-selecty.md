# How to use PageFly Landing Page Builder with Selecty

  

# ✔️ About PageFly Landing Page Builder

  

[PageFly Landing Page Builder](https://apps.shopify.com/pagefly) is a drag-and-drop editor that allows merchants to create and customize attractive pages for their stores.

  

The app provides multiple effective options for creating pages:

  

*   You can select among the 6 main page types or use 100+ templates.
*   CRO-focused elements like a countdown timer, sales banner, cross-selling, color swatches, etc.
*   Various options and blocks for customization.

  

With the help of PageFly, you can create different types of pages: landing, Home page, Product and Collection pages, Blog, Help Center, etc. Furthermore, you can use the Selecty to add language and currency selectors to the created pages.

  

# ✔️ Install PageFly Landing Page Builder

  

So, before you start creating pages with language & currency selectors, please install both of the applications:

  

*   [PageFly Landing Page Builder](https://apps.shopify.com/pagefly)
*   [Geolocation & Markets - Selecty](https://apps.shopify.com/selectors)

  

# ✔️ Create a new page with PageFly and connect Selecty

  

****1\. Create a product page with PageFly****

  

Open PageFly and click ****Create your first page**** -> select page type. For example, let’s create a Regular page.

  

![](https://cdn.heymantle.com/docs/screenshots/bb7ce538-9bba-44ba-8533-d3426fb59ddf/create-your-first-page_amez2i.png)

  

In the left-side menu, you will find available blocks, elements, and customization features. Use them to create your first page. The app also provides various page templates, so you can use one of them and customize it if needed.

  

****2\. Add and activate the Selecty element****

  

First of all, you need to add the Selecty app to the PageFly and activate it.

  

*   Go to the ****Third-party elements section**** and click on the Third-party integrations button.
*   In the ****Integrations****, search for Geolocation & Markets - Selecty.
*   Click ****Activate**** -> click ****Done**** to show the element in the catalog.

  

![](https://cdn.heymantle.com/docs/screenshots/8b99a85a-adb6-4503-8f23-e96c71ac4e39/pagefly-selector-app_1r7zbwl.png)

  

****3\. Add a selector to the page****

  

*   In the left-side menu click on the ****Third-party integrations**** button.
*   Search for Selecty and click on it -> you will see available variants of selectors.
*   Drag and drop the selected variant to the page.

  

****4\. Create the selector in the Selecty app****

  

Go to the Selecty app and click ****Create selector**** -> Select the Theme -> make the appropriate settings in the right-side menu -> ****Save**** and ****Publish**** it.

  

****5\. Customize the selector in PageFly****

  

Now you can customize your selector and you can do it not leaving PageFly.

  

Click on the selector widget on your page. In the right-side menu you will see available customization options. For example, in the General tab you can configure the selector’s content, visibility, animation, etc. The Styling tab provides you with style settings like spacing, border style, opacity effect, CSS editor, etc.

  

![](https://cdn.heymantle.com/docs/screenshots/7f8df0f4-8bc3-4d99-b794-12a988b4177f/pagefly-customize-selector_q9eue4.png)

  

Note that you can customize your selector inside the Selecty app. To learn more about the customization, please check our article – [Can I customize selectors?](/can-i-customize-selectors)

  

****6\. Publish your page****

  

After you have made all the settings and are satisfied with the result, ****Save**** the page and publish it by clicking the ****Publish**** button in the upper right corner.

  

# ✔️ Benefits of using PageFly

  

*   ****No coding required**** – you don’t need any technical background to build a professionally-looking website. The app provides drag-and-drop functionality that allows you to create pages without coding.
*   ****Flexible design options & templates**** – PageFly allows you to add and customize all types of page content. You can also use pre-designed page templates and configure them to fit your brand style.
*   ****Time-saving**** – the builder significantly reduces the time required to create and launch a website. Edit your page in real-time and see the result immediately.
*   ****Integration capabilities**** – PageFly includes built-in SEO tools and is compatible with different third-party apps, namely, with Selecty.
