# About PageFly Landing Page Builder

[PageFly Landing Page Builder](https://apps.shopify.com/pagefly) is a drag-and-drop editor that allows merchants to create and customize pages for their stores.

The app provides multiple options for creating pages:

*   Choose from 6 main page types or use 100+ templates.
*   CRO-focused elements such as a countdown timer, sales banner, cross-selling, and color swatches.
*   A wide range of customization options and blocks.

With PageFly, you can create landing pages, home pages, product and collection pages, blogs, help centers, and more. You can also use the language and currency selectors on pages you create.

## Install PageFly Landing Page Builder

Before creating pages with language and currency selectors, install both applications:

*   [PageFly Landing Page Builder](https://apps.shopify.com/pagefly)
*   [Geolocation & Markets](https://apps.shopify.com/selectors)

## Create a new page with PageFly and add a selector

**1. Create a product page with PageFly**

Open PageFly and click **Create your first page** → select a page type. For example, let's create a Regular page.

![](https://cdn.heymantle.com/docs/screenshots/bb7ce538-9bba-44ba-8533-d3426fb59ddf/create-your-first-page_amez2i.png)

In the left-side menu, you will find available blocks, elements, and customization features. Use them to build your page. The app also provides page templates that you can customize as needed.

**2. Add and activate the selector element**

First, add the app to PageFly and activate it.

*   Go to the **Third-party elements** section and click **Third-party integrations**.
*   In **Integrations**, search for Geolocation & Markets.
*   Click **Activate** → click **Done** to show the element in the catalog.

![](https://cdn.heymantle.com/docs/screenshots/8b99a85a-adb6-4503-8f23-e96c71ac4e39/pagefly-selector-app_1r7zbwl.png)

**3. Add a selector to the page**

*   In the left-side menu, click **Third-party integrations**.
*   Search for the selector app and click on it → you will see the available selector variants.
*   Drag and drop your chosen variant onto the page.

**4. Create the selector in the app**

Go to the app and click **Create selector** → select a theme → configure the settings in the right-side menu → **Save** and **Publish**.

**5. Customize the selector in PageFly**

You can customize your selector without leaving PageFly.

Click on the selector widget on your page. The right-side menu shows available customization options. For example, the **General** tab lets you configure content, visibility, and animation. The **Styling** tab provides style settings such as spacing, border style, opacity, and a CSS editor.

![](https://cdn.heymantle.com/docs/screenshots/7f8df0f4-8bc3-4d99-b794-12a988b4177f/pagefly-customize-selector_q9eue4.png)

You can also customize the selector from within the app itself. To learn more, see [Can I customize selectors?](/selecty/can-i-customize-selectors)

**6. Publish your page**

Once you are satisfied with the result, **Save** the page and click **Publish** in the upper right corner.

## Benefits of using PageFly

*   **No coding required** — you do not need a technical background to build a professional-looking page. The drag-and-drop interface makes it accessible to everyone.
*   **Flexible design options and templates** — add and customize all types of page content, or start from a pre-designed template and adjust it to fit your brand.
*   **Time-saving** — the builder significantly reduces the time needed to create and launch a page. Edit in real time and see results immediately.
*   **Integration capabilities** — PageFly includes built-in SEO tools and is compatible with third-party apps, including this one.
