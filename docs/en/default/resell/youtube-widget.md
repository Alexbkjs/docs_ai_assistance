# YouTube widget

In this article:

*   [Add widget](#1-add-widget)

Add a ****YouTube video**** to your offer to showcase products, tutorials, or brand stories. Engage visitors visually and boost trust through authentic video content.

Available on Checkout, Post-purchase, Thank you page, and Order status pages.

# ✔️ Add widget

1.  Open the funnel and move to the step where you want to add the widget.
2.  Click ****Edit offer****.
3.  In the left side menu, click ****Add widget**** -> YouTube widget.
4.  In the right menu, make the appropriate settings: set the type of content, set text, minimal amount, and colors.

![](https://cdn.heymantle.com/docs/screenshots/58705c82-caab-4498-bb58-2b18ae1cbb2c/you-tube-video-widget_yqakw1.png)

5.  Click ****Save**** in the upper right corner.
