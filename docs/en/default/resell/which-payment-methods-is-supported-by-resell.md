# Supported Payment Methods

## Post-Purchase offers

Shopify imposes some restrictions. The Post-purchase page is available on Shopify and works correctly with the following payment methods:

*   Shopify Payments
*   [PayPal Express](https://help.shopify.com/en/manual/payments/paypal) (with Automatic Payments enabled, known by PayPal as Reference Transactions)
*   [Shop Pay](https://www.shopify.com/blog/shop-pay-checkout)
*   PayPal Pro, PayPal PayFlow (when used with a credit card)
*   Braintree (without CVV verification enabled)
*   Stripe

There are also limitations that prevent post-purchase offers from working. You can learn more in this article: [Why is my post-purchase upsell offer not displayed?](/resell/why-is-my-post-purchase-upsell-offer-not-displayed)

## Checkout, Thank You page, Order Status page

The Checkout, Thank you page, and Order Status pages do not depend on the payment method. There are no limitations for these stages.

Please note that Shopify only supports vaulted card payment processors. For more information, refer to this article: [Vaulted payment processors](/resell/vaulted-payment-processors).
