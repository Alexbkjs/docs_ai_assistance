# Edit offers

ReSell is a flexible app with extensive customization features. To edit a post-purchase offer, click **Edit offer** next to the selected product. You will be taken to the customization page for that offer.

![](https://cdn.heymantle.com/docs/screenshots/fa107b65-a2a6-49b7-9ed5-fd8dd069f81b/edit-offer_1sohf93.png)

## Edit existing elements

Click on any element you want to edit. In the right-side panel that opens, you can change the title, content, text size and alignment, and toggle the border and background on or off.

![](https://cdn.heymantle.com/docs/screenshots/e140bf11-62f8-407f-85b4-8f4eaf8f9dbb/edit-element_1uvlm8w.png)

The **Product** section provides an extended set of settings. Here you can change the product, add a custom description, show or hide product variants and tax lines, apply a discount, and change the main product image.

## Add new elements

To add new elements or widgets to your offer, click **Add element** and select the widgets you need.

![](https://cdn.heymantle.com/docs/screenshots/8a9f34ad-d83f-41b5-8a16-0b816ebb289e/add-new-widget_1jprump.png)
