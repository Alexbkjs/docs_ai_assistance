# Trust badge

Use the **Trust badge widget** to display security and reliability badges that reassure your customers. Available on Checkout, Post-purchase, Thank you page, and Order status pages.

## Add widget

1.  Open the funnel and move to the step where you want to add the widget.

2.  Click **Edit offer**.

3.  In the left side menu, click **Add widget** -> Trust badge.

4.  In the right menu, configure the appropriate settings: set the content type, text, minimum amount, and colors.

5.  Click **Save** in the upper right corner.
