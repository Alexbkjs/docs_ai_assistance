# Trust badge

Show security and reliability badges to reassure your customers with the ****Trust badge widget****. Available on Checkout, Post-purchase, Thank you page, and Order status pages.

  

# ✔️ Add widget

  

1.  Open the funnel and move to the step where you want to add the widget.
2.  Click ****Edit offer****.
3.  In the left side menu click ****Add widget**** ->Trust badge.
4.  In the right menu make the appropriate settings: set the type of content, set text, minimal amount, and colors.

  

  

5.  Click ****Save**** in the upper right corner.
