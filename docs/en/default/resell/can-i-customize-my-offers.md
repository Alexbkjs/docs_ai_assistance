# Customization Tools

Yes! ReSell has a built-in editor with a wide range of widgets for customizing post-purchase and Thank you page offers. No coding skills are required. You can customize all offers to match the design of your store.

To edit an offer, click **Edit offer** in the upper-right corner of the offer's block. The editor will open, showing all available customization tools.
