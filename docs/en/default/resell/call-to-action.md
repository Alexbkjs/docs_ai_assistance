# Call to action banner

  

In this article:

  

*   [Add widget](#1-add-widget)
*   [Advanced setup](#1-advanced-setup)

  

With the ****call to action banner**** you can encourage customers to take action — complete a purchase, explore a deal, or sign up.

  

Available on Checkout, Post-purchase, Thank you page, and Order status pages.

  

# ✔️ Add widget

  

1.  Open the funnel and move to the step where you want to add the widget.
2.  Click ****Edit offer****.
3.  In the left side menu click ****Add widget**** -> Call to action.
4.  In the right menu make the appropriate settings: set the type of content, set text, minimal amount, and colors.

  

![](https://cdn.heymantle.com/docs/screenshots/7d09e25c-e326-4ec8-8fdd-053e2a89d1cc/call-to-action_1922gbv.png)

  

5.  Click ****Save**** in the upper right corner.
