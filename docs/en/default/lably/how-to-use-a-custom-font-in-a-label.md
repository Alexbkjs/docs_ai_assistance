# How to use a custom font in a label?

# Add custom font

You can use any font that is already included in the theme. You can also connect a custom font that you need.

To add a custom font, follow these steps:

1.  Open the label where you want to add a custom font.
2.  In the left-side menu, open the ****Styles**** section.
3.  Enable the ****Use custom font**** option.
4.  In the appeared text field, enter the font family.
5.  Click the ****Save**** button in the upper-right corner.

You can read more about fonts in the Shopify article – [<u><span style="white-space:pre-wrap" class="csh-markdown csh-markdown-underline">Custom Fonts</span></u>](https://shopify.dev/docs/storefronts/themes/architecture/settings/fonts#custom-fonts).
