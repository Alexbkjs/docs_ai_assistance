# ✔️ About the Lably app

The [Lably - Product Labels & Badges](https://apps.shopify.com/lably) app allows you to add eye-catching, customizable badges and labels to your products, highlight particular products, and motivate customers to make more purchases in your store.

Note that ****labels**** will be displayed on the product's image, and ****badges**** will be displayed in product's content.

# ✔️ Steps to add your first label or badge

1.  Enter the app and click on ****Add new****.
2.  Here you can select the option: label or badge. Let's choose ****Add new label****.

# ✔️ Customization features

The app provides multiple customization options which allow you to customize and display the badges as you need.

In the left-side menu, you can see all available options. Let us briefly describe each of them.

![](https://cdn.heymantle.com/docs/screenshots/1f4ea351-fff2-463e-b980-43b2232619c4/label-customization-options_juukw9.png)

*   ****Label / Badge**** - in this section, you need to choose the type of your label or badge:
    
*   Select an image and upload your image or use pre-designed thematic product labels.
    
*   Select the Text type to set the appropriate text in the text field.
    

Note: You can [add translation](/localization) to your badges and labels.

![](https://cdn.heymantle.com/docs/screenshots/721d3914-5657-4261-9c3c-4721ee6bd933/label-type-text_1b4i3v7.png)

*   ****Styles**** - here you can customize your label. For example, set its shape, font, size, style, etc.
*   ****Position**** - set the label position and margin.
*   ****Link**** - enter the link on your label if you want to move customers to the particular product or page when they click on the label.
*   ****Animations**** - here you can select the animation type of your label.
*   ****Week schedule**** - select the day and time when you want to display your label in your store. By default, all options are enabled. This means that the label will be displayed all the time until you delete it. You can set the appropriate time for each day separately.
*   ****Visibility on pages**** - select pages where you want to display labels on your product. For example, if you select Home page -> the product label will be displayed only on the Home page. When the customer clicks on it and moves to the product page, the label will disappear.
*   ****Display period**** - in this section, you can plan the display of the label. Just disable “All time” and select the period when you want to display it.
*   ****Display conditions**** - this option allows you to add labels to particular products.
