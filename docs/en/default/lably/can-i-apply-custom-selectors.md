# Custom selectors

In some themes, labels may not be displayed. In these cases, you can use a custom selector to ensure they appear correctly.

## Apply a custom selector

Follow these steps to apply a custom selector:

1.  Open the badge or label and go to **Position**.
2.  Find **Custom CSS selector** in the left-side menu.
3.  Enter the code of the element where you want to display the badge or label.

## Where to get the code

1.  Open your store and find the product (or other element) where you want to display the selector.
2.  Right-click on it and select **Inspect**.
3.  Copy the relevant part of the element's code and paste it into the **Custom CSS** field described above.

**Note:** Read more about [<u><span style="white-space:pre-wrap" class="csh-markdown csh-markdown-underline">CSS selectors</span></u>](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_selectors).

## Troubleshooting

If you completed all the steps above but the selector is still not displaying, try the following:

*   Check that the app is enabled in App embeds.
