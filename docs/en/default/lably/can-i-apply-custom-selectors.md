# Can I apply custom selectors?

# Custom selectors

In specific themes, labels may not be displayed. In such cases, custom selectors are used to display them correctly. 

  

# ✔️ Apply a custom selector

  

Here is a step-by-step instruction on how to apply a custom selector:

  

1.  Open the badge/label and go to ****Position****.
2.  Find the ****Custom CSS selector**** on the left-side menu.
3.  Enter the code of the element where you want to display the badge/label.

  

# ✔️ Where to get the code?

  

1.  Open your store and find the product (or another element) where you want to display the selector.
2.  Right-click on it and click ****Inspect****.
3.  Copy the part of the element’s code and paste it in the ****Custom CSS**** field that was mentioned in the instructions above.
