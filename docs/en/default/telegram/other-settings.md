# ✔️ Manage username display

You can manage username display by using the ‘Show names of Discord users in Telegram’. This option allows you to customize the display of messages from the bot in Telegram.

By default, it’s disabled and your messages will be displayed as usual messages from the bot. But for example, you have a support channel, and you want your messages to display with text like “New message from John”.

To activate this option, follow these steps:

1.  Use the `/list` command to bring up a modal window with settings.
2.  Click on Settings -> Choose the 'Other settings' -> Show names of Discord users in Telegram.
3.  Click ****Enable**** -> Edit message -> enter the text you want to display -> click Submit.

![](https://cdn.heymantle.com/docs/screenshots/1af425b8-3bdd-4c75-ad2f-703cc39b70e9/other-settings_q72yy.png)

You can use the following variables to customize your welcome message:

*   `{{displayName}}`: Displays the user's full name (e.g., John Smith).

![](https://cdn.heymantle.com/docs/screenshots/6858a54e-f04b-4913-a4f6-58accf804f8f/notification-settings-result_hp24ce.png)

This feature allows you to personalize messages, making them appear as if they are coming from a real person rather than a bot. Personalized messages help strengthen customers engagement, enhance user trust, and create a more natural and interactive communication experience.

# ✔️ Re-create conversation thread

This option automatically generates a new thread when the current one remains inactive for an extended period. You can specify the time period after which a new thread will be created.

To activate this option, follow these steps:

1.  Use the `/list` command to bring up a modal window with settings.
2.  Click on Settings -> Choose the 'Other settings'.
3.  Find the button ****‘Re-create conversation thread after’**** and select the time period when you want to re-create your thread: in 24 hours or 1 week. Selecting ‘Never re-create’ prevents new threads, even after long inactivity.
