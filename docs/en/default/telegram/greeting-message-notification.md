# Manage Greeting Notifications

When a user sends the `/start` command, the bot can automatically reply with a greeting message and a request for additional details. You can fully customize this message to match your brand's tone and communication style.

To activate this option, follow these steps:

1.  Use the `/list` command to open the settings modal.
2.  Click on Settings → choose a settings category → **Greeting Message Notification**.
3.  Click **Enable** → **Edit message**. In the modal window that appears, enter the text you want to use as the greeting message.

![](https://cdn.heymantle.com/docs/screenshots/e665f9ee-81a8-4726-b0f9-fd06dc110ba4/greeting-message-option_136vv2o.png)

When users send the `/start` command, they will receive your personalized greeting, improving their engagement and overall experience.
