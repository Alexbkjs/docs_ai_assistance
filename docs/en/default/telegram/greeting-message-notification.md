# Manage Greeting Notifications

When a user presses the `/start` command, the bot can automatically send a greeting message and a request for additional details. You can fully customize this message to align with your brand’s tone and communication style.

To activate this option, follow these steps:

1.  Use the `/list` command to bring up a modal window with settings.
2.  Click on Settings -> Choose the settings -> ****Greeting Message Notification****.
3.  Click ****Enable**** -> Edit message. In the appeared modal window, you can enter the text you want to display as a greeting message.

![](https://cdn.heymantle.com/docs/screenshots/e665f9ee-81a8-4726-b0f9-fd06dc110ba4/greeting-message-option_136vv2o.png)

As a result, when users send the `/start` command, they will receive a personalized and welcoming greeting message, enhancing their engagement and overall experience.

![](https://cdn.heymantle.com/docs/screenshots/66a3bdcd-70a3-48ee-94ee-4c2e6b7c43d5/greeting-message-setup-and-exa_ifwash.png)
