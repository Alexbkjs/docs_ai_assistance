# Forwarding Directions

This option allows you to set up one-way message forwarding between Telegram and Discord. For example, you can enable message forwarding from Telegram to Discord, but disable message forwarding from Discord to Telegram. This will allow you to control the flow of messages, leaving it one-way only.

To activate this option, follow these steps:

1.  Use the `/list` command to bring up a modal window with settings.
2.  Click on Settings -> Forwarding Directions.
3.  Click Enable near the option you want to activate.

![](https://cdn.heymantle.com/docs/screenshots/4af9945e-cba9-4fce-941d-6028dfc8834e/directions_2tdh9a.png)

By default both options are enabled that allows forwarding messages from Telegram to Discord and backward.

Note, that if you select the "Forward from Telegram to Discord: Disabled" option, a thread or channel will only be created after a message is sent from Telegram. The message itself will not appear in the created channel or thread. It will only create a thread or channel. Once a thread or channel is created, you can write there and messages will only flow one-way from Discord to Telegram.

*   [Message Filters](/message-filters)
*   [Greeting Message Notification](/greeting-message-notification)
