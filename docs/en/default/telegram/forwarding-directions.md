This option allows you to set up one-way message forwarding between Telegram and Discord. For example, you can enable forwarding from Telegram to Discord while disabling forwarding from Discord to Telegram, giving you full control over the message flow.

To activate this option, follow these steps:

1.  Use the `/list` command to open the settings modal.
2.  Click on Settings → Forwarding Directions.
3.  Click **Enable** next to the option you want to activate.

![](https://cdn.heymantle.com/docs/screenshots/4af9945e-cba9-4fce-941d-6028dfc8834e/directions_2tdh9a.png)

By default, both directions are enabled, allowing messages to flow from Telegram to Discord and back.

Note: If you select "Forward from Telegram to Discord: Disabled", a thread or channel is created only after a message is sent from Telegram — but that message will not appear in the created channel or thread. Once the thread or channel exists, messages will flow one-way from Discord to Telegram only.
