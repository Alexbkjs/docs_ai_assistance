# Message Filters

# Manage Message Filters

In some cases, users must press the Start button or send the `/start` command to initiate a conversation with the bot, depending on your settings. By default, this command is forwarded to Discord.

This option allows you to ignore the `/start` command, preventing it from being sent to Discord and ensuring a cleaner message flow.

To activate this option, follow these steps:

1.  Use the `/list` command to bring up a modal window with settings.
2.  Click on Settings -> Choose the settings -> ****Message**** ****Filters****.
3.  Click ****Enable**** near the 'Ignore “/start” message' button.

![](https://cdn.heymantle.com/docs/screenshots/d6fcc2c1-e85d-40a4-8242-26db840fd6b8/message-filters-settings_1iey2u6.png)

Now, the `/start` command won’t be forwarded to Discord.

![](https://cdn.heymantle.com/docs/screenshots/6324f491-35bd-47f9-87d8-75f13efe0fa5/m-filter_1jiuyer.png)
