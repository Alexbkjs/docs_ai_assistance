# Manage Message Filters

In some cases, users must press the **Start** button or send the `/start` command to initiate a conversation with the bot, depending on your settings. By default, this command is forwarded to Discord.

This option allows you to ignore the `/start` command so it is not sent to Discord, keeping your message flow clean.

To activate this option, follow these steps:

1.  Use the `/list` command to open the settings modal.
2.  Click on Settings → choose a settings category → **Message Filters**.
3.  Click **Enable** next to the 'Ignore "/start" message' option.

![](https://cdn.heymantle.com/docs/screenshots/d6fcc2c1-e85d-40a4-8242-26db840fd6b8/message-filters-settings_1iey2u6.png)

The `/start` command will no longer be forwarded to Discord.

![](https://cdn.heymantle.com/docs/screenshots/6324f491-35bd-47f9-87d8-75f13efe0fa5/m-filter_1jiuyer.png)

In the Settings section, you can also customize:

*   [New Message Notification](/telegram/new-message-notification)
*   [Message Forwarding](/telegram/message-forwarding)
*   [Greeting Message Notification](/telegram/greeting-message-notification)
*   [Other Settings](/telegram/other-settings)
