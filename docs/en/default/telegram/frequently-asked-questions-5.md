# ✔️ What if a customer replies to an old thread?

If the thread is more than 24 hours old, a new thread will be created. Otherwise, the customer's reply will be added to the existing thread.

# ✔️ Does your app store messages?

No, our application doesn't store your messages.

# ✔️ If my customer uses an attachment in the message, will I see it in my Discord channel?

Yes. All messages will be redirected with attached files, images, emoji, etc.

# ✔️ Why my Telegram avatar is not displayed in Discord?

Check your Telegram settings → Privacy and Security → Profile photo. It should be either set to «Everybody» or allowed in exceptions for group or bot.

# ✔️ How to remove integration?

Use `/disconnect` command in any Text channel.

# ✔️ If I have questions about the app, how can I get help?

You can get it in 2 ways:

*   Type the `/help` command in the text field and press ****Enter****. You will get a short guide and contacts of our support.
*   Directly contact our support team: [support@devit.software](mailto:support@devit.software).

# ✔️ Where can I leave feedback about the app?

You can leave your feedback on the [Telegram bot page](https://top.gg/bot/1123949801613049866).

# ✔️ Why is Full Name displayed instead of Username?

In Telegram, Full Name (or more specifically, "First Name") is a mandatory field, whereas "Username" is optional, meaning not all users have one. If a user has set a username, it will be displayed when using the variables {{username}} or {{name}}. However, if no username is available, Telegram will display the user's first name by default.
