# FAQ

## What if a customer replies to an old thread?

If the thread is more than 24 hours old, a new thread will be created. Otherwise, the customer's reply is added to the existing thread.

## Does your app store messages?

No, the application does not store your messages.

## If my customer uses an attachment in the message, will I see it in my Discord channel?

Yes. All messages are forwarded with attached files, images, emoji, and so on.

## Why is my Telegram avatar not displayed in Discord?

Check your Telegram settings → Privacy and Security → Profile photo. It should be set to "Everybody" or allowed in exceptions for the group or bot.

## How do I remove the integration?

Use the `/disconnect` command in any text channel.

## If I have questions about the app, how can I get help?

There are two ways to get help:

*   Type the `/help` command in the text field and press **Enter**. You will receive a short guide and our support contact details.
*   Contact our support team directly: [support@devit.software](mailto:support@devit.software).

## Where can I leave feedback about the app?

You can leave feedback on the [Telegram bot page](https://top.gg/bot/1123949801613049866).

## Why is Full Name displayed instead of Username?

In Telegram, Full Name (specifically "First Name") is a required field, while "Username" is optional — not all users have one. If a user has set a username, it will be displayed when using the `{{username}}` or `{{name}}` variables. If no username is set, Telegram displays the user's first name by default.
