# Set Message Forwarding

This option allows you to choose where to forward messages from Telegram. You can send messages from Telegram groups, channels, and users to Discord channels or threads.

1.  Use the `/list` command to open the settings modal.
2.  Click on Settings → choose a settings category → **Message Forwarding**.

Here you can configure where messages from a Telegram channel or group are forwarded in Discord. By default, **Threads** is enabled, so messages from Telegram groups and channels are forwarded to threads.

![](https://cdn.heymantle.com/docs/screenshots/faedd5a5-b33d-4514-a410-865a3e480c1a/message-forwarding-option_1bm92lx.png)

To change this, click **Channels**. Messages will then be forwarded to a designated channel instead.

![](https://cdn.heymantle.com/docs/screenshots/c7cf166f-400a-4d48-9c7c-afc091ffd4e8/message-fprwarding-channels_1c34ouc.png)

In the Settings section, you can also customize:

*   [New Message Notification](/telegram/new-message-notification)
*   [Message Filters](/telegram/message-filters)
*   [Greeting Message Notification](/telegram/greeting-message-notification)
*   [Other Settings](/telegram/other-settings)
