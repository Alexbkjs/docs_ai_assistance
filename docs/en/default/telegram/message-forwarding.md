# Message Forwarding

  

# Set Message Forwarding

  

This option allows you to select where to forward messages from Telegram. Namely, you can send messages from Telegram groups, channels, and users to Discord channels and threads.

  

1.  Use the `/list` command to bring up a modal window with settings.
2.  Click on Settings -> Choose the settings -> ****Message Forwarding****.

  

Here you can configure where messages from a Telegram channel or group should be forwarded to Discord. By default, Threads are enabled and messages from Telegram groups and channels will be forwarded to threads.

  

![](https://cdn.heymantle.com/docs/screenshots/faedd5a5-b33d-4514-a410-865a3e480c1a/message-forwarding-option_1bm92lx.png)

If you wish to modify this setting, simply click the Channels button. This ensures that when users send messages, they will be forwarded to a designated channel.

  

![](https://cdn.heymantle.com/docs/screenshots/c7cf166f-400a-4d48-9c7c-afc091ffd4e8/message-fprwarding-channels_1c34ouc.png)
