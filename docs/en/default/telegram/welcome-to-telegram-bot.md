## What is a Telegram bot?

[Telegram bot](https://top.gg/bot/1123949801613049866) allows you to forward Telegram chats to a connected Discord channel and use it as your customer support bot.

**Main features**

*   Each chat from the Telegram bot appears in a new thread or channel in Discord.
*   Real-time seamless messaging between your team and customers, with matching avatars.
*   Create a new bot or use an existing one.
*   The bot supports documents, photos, videos, voice messages, audio, stickers, emoji, and avatars.
*   Multiple settings allow you to configure the app according to your needs.

## How does it work?

You create a bot in Telegram (or use an existing Telegram bot) → get a token → provide the Telegram bot token using the `/connect` command in the Discord channel.

This forwards messages from your Telegram bot to the connected Discord channel as public threads or channels.

All replies in the created threads are forwarded back to the Telegram chat.

![](https://cdn.heymantle.com/docs/screenshots/7cbc2767-eb31-4353-8216-dfc1f88be3b0/telegram-bot_k6zda2.png)

To set up the application according to your needs, use the **Settings** section. Send the `/list` command in the channel where you connected the app, then click **Settings**.

In the Settings section, you can also customize:

*   [New Message Notification](/telegram/new-message-notification)
*   [Message Forwarding](/telegram/message-forwarding)
*   [Message Filters](/telegram/message-filters)
*   [Greeting Message Notification](/telegram/greeting-message-notification)
*   [Other Settings](/telegram/other-settings)
