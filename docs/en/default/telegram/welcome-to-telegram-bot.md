# ✔️ What is a Telegram bot?

  

[Telegram bot](https://top.gg/bot/1123949801613049866) allows you to forward Telegram chats to the connected Discord channel and use it as your customer support bot.

  

****Main features****

  

*   Each chat from the Telegram bot appears in a new thread or channel in Discord.
*   Real-time seamless messaging between your team and customers with the same avatars.
*   Create a new bot or use an existing one.
*   Bot supports document, photo, video, voice, audio, stickers, emoji, avatars.
*   Multiple setting allows to setup the app according to your needs.

  

# ✔️ How does it work?

  

You create a bot in Telegram messenger (or use existed Telegram bot) -> get token -> provide Telegram bot token with the `/connect` command in the Discord channel.

  

Thus, you will forward messages from Telegram bot to the connected Discord channel in public threads or channels.

  

All your replies in created threads will be forwarded back into a Telegram chat.

  

![](https://cdn.heymantle.com/docs/screenshots/7cbc2767-eb31-4353-8216-dfc1f88be3b0/telegram-bot_k6zda2.png)

  

To setup the application according to your needs, you can use the ****Settings**** section. Send the `/list` command in the channel where you've just connected the app, and click Settings.
